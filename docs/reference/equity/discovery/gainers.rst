Gainers
=======

Get the top price gainers in the stock market.

Example:

Use ``equity.discovery.gainers()`` with default parameters: ``provider`` (vndirect), ``exchange`` (hose).

.. code-block:: python

    from vietfin import vf
    
    # Get the top price gainers in the stock market, in HOSE exchange, from provider VnDirect
    vf.equity.discovery.gainers()
